Onboarding Lab: Access and Flags
Platform: HackerDNA
Difficulty: Very Easy
Objective: Learn how to navigate the platform, access the target, locate flags, and submit them to complete the lab.

Steps
Start the machine in HackerDNA.

Click the provided lab URL (example: https://lab.hdna.me/21-learn-101).

On the welcome screen, click Access the Flag.

A flag will appear on a blank page.

Copy the flag exactly as shown.

Return to the lab interface and paste the flag into the User Flag field.

Click Submit.

Login Portion
On the login page, press F12 to open Developer Tools.

Go to Sources → script.js.

Inside the script, locate:

Username

Password

Flag reference

Log in using the credentials found in the script.

Enter the flag shown in the script or referenced file.

Click Submit to complete the lab.

Key Takeaways
This lab teaches basic navigation of the HackerDNA platform.

Flags must be copied exactly as displayed.

Developer Tools reveal client‑side scripts, credentials, and flag references.

Understanding how to inspect scripts prepares you for later labs.
